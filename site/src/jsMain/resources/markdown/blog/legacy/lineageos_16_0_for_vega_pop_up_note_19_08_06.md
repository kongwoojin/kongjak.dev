# LineageOS 16.0 for Vega Pop-Up Note 19/08/06

**개발자는 책임이 없음을 알립니다.**

작동
- 와이파이
- 블루투스
- GPS
- 소리

- V Pen은 강하게 눌러줘야 인식됩니다.
